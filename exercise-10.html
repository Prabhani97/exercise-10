Step 4

Using a permutation approach,

First I need to inference to generate a null distribution of F statistic values from the model of log(Mass) in relation to Trophic.Level.

So I am going to fit original model and get observed F staistic like above,

observed_F <- aov(log(Mass) ~ Trophic.Level, data = d) %>%
  broom::tidy() %>%
  filter(term == "Trophic.Level")
observed_F

Now from the methid I know I can simulate no relationship by shuffling,

d_perm <- d   # creating a copy of the dataset

d_perm$Trophic.Level <- sample(d$Trophic.Level)

Recomputing F each time,

aov(log(Mass) ~ Trophic.Level, data = d_perm)
Call:
   aov(formula = log(Mass) ~ Trophic.Level, data = d_perm)

                Trophic.Level Residuals
Sum of Squares         11.175 26592.443
Deg. of Freedom             3     11000

Residual standard error: 1.55483
Estimated effects may be unbalanced

Repeating 1000 times now,

nperm <- 1000
permuted.F <- vector(length = nperm)

for (i in 1:nperm) {
  d_perm <- d
  d_perm$Trophic.Level <- sample(d$Trophic.Level)

  permuted.F[i] <- aov(log(Mass) ~ Trophic.Level, data = d_perm) %>%
    broom::tidy() %>%
    filter(term == "Trophic.Level") %>%
    pull(statistic)
}
p.value <- mean(permuted.F >= observed_F$statistic)
p.value
[1] 0

What I received is Real F = 78.22 Permutation p = 0.

I did above assuming that there is no correlation between log(Mass) and Trophic.Level. None of the permuted F values were as large as the observed F statistic after this procedure was repeated 1000 times. This produced a permutation based p value of almost zero, telling me that the observed association is not by chance.
